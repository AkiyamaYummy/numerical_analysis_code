least_squares.cpp为题目的解题源代码，程序中指明了in.txt为程序的输入文件，其中包含程序的输入数据。
输入数据可包含多组，每组数据的数据格式为：
第一行包含两个整数n和mode。n代表(t,y)数据组数。mode选择了近似表达式的种类。mode为0时，近似表达式为f(t)=a1*t+a2*t^2+a3*t^3，mode为1时，近似表达式为f(t)=a0+a1*t。
第二行包含n个实数，代表n个t的取值。
第三行包含n个实数，代表对应的n个y的取值。
第四行包含n个实数，代表n组(t,y)对应的n个权值p。
每组程序输出包含两行。第一行为拟合曲线的函数。第二行为使用该拟合曲线后的均方误差。
