gauss.cpp为题目的解题源代码，程序中指明了in.txt为程序的输入文件，其中包含程序的输入数据。
输入数据可包含多组，每组数据的数据格式为：
第1行包含两个整数，n和mode，n为未知数个数，mode指定了选主元的策略。mode为0则为顺序高斯消去法，为1则每次选取模最大的主元，为2则每次选取模最小的非0主元，为3则为手动选取主元。
第2行到第n+1行，每行有n+1个实数，即为输入了一个n*(n+1)的矩阵，代表[A|y]。
输入完毕并计算结果后，程序会输出两行数据，第一行中含有一个实数，代表改系数矩阵A的1条件数。第二行含有n个实数，依次为该方程组的解。
如果方程组使用对应的选主元策略无法解出，程序会输出“error”。
特别地，在手动选取主元的模式下，每次选取主元会输出一行备选项，格式如下：
编号1:列元1 编号2:列元2 ... 编号m:列元m
如果决定选用某个列元为主元，输入列元对应的编号并回车即可。
